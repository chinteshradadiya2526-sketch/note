Java Notes

Introduction

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. It is widely used for building enterprise-scale web applications, Android apps, and more.

Key Features

  • Simple and easy to learn
  • Object-Oriented
  • Platform Independent (Write Once, Run Anywhere)
  • Robust and Secure
  • Multithreaded
  • Rich API and vast community support

Basic Syntax

  • Java programs are made up of classes and methods.
  • The main method is the entry point of any Java application.

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Object-Oriented Concepts

  • Class: Blueprint for objects.
  • Object: Instance of a class.
  • Inheritance: Mechanism for a new class to inherit properties of another class.
  • Polymorphism: Ability to take many forms.
  • Encapsulation: Wrapping data and code together.
  • Abstraction: Hiding internal details and showing functionality.
